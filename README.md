# 🛵 OpenMarket

منصة توصيل طلبات شاملة للمنازل - مدينة الإسكندرية، مصر

---

## 📦 عن المشروع

OpenMarket هي منصة توصيل طلبات تربط بين:
- **العملاء** (المستخدمين)
- **المتاجر** (مطاعم، مخابز، صيدليات، فواكه وخضروات، منتجات محلية)
- **السائقين** (للتوصيل)

---

## 🗄️ هيكل قاعدة البيانات

| # | الجدول | الوصف |
|---|--------|-------|
| 1 | `users` | العملاء |
| 2 | `drivers` | السائقين |
| 3 | `driver_locations` | تتبع مواقع السائقين |
| 4 | `zones` | المناطق ورسوم التوصيل |
| 5 | `stores` | المتاجر |
| 6 | `categories` | فئات المنتجات |
| 7 | `store_hours` | ساعات العمل |
| 8 | `products` | المنتجات |
| 9 | `product_options` | خيارات المنتج |
| 10 | `option_values` | قيم الخيارات |
| 11 | `product_addons` | إضافات المنتج |
| 12 | `addresses` | عناوين التوصيل |
| 13 | `carts` | عربة التسوق |
| 14 | `orders` | الطلبات |
| 15 | `order_items` | عناصر الطلب |
| 16 | `payments` | المدفوعات |
| 17 | `reviews` | التقييمات |
| 18 | `coupons` | أكواد الخصم |
| 19 | `coupon_usages` | استخدامات الخصم |
| 20 | `notifications` | الإشعارات |

---

## 🛠️ التقنيات المستخدمة

- **قاعدة البيانات:** PostgreSQL (Supabase)
- **المفاتيح الأساسية:** UUID
- **المواقع الجغرافية:** PostGIS
- **اللغة:** SQL

---

## 📍 الموقع الجغرافي

- **المدينة:** الإسكندرية، مصر
- **نطاق التوصيل:** جميع أحياء الإسكندرية
